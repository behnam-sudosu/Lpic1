# uname

	uname ===>> show what you use distro
	uname -a ===>>  Show kernel infomation
	uname -r ===>> kernel release
	uname -s ===>> kernel name
	uname -p ===>> kernel architetor
	uname -i ===>> hardware platform
	uname -n ===>> hostname or nodename
	uname -v ===>> version of kernel
	uname -m ===>> print the machine's hardware (cpu name)
	uname -o ===>> print the operating system name
