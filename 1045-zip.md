# zip

### zip

```bash
zip ===>> multiplatform compressing tool, both archiving and compressing
zip ===>> compress files and folders
zip file.zip file1
unzip file1 ===>> decompress files and folders
```

---

### gzip

```bash
gzip ===>> great tool for compression
gzip file1 ===>> Compresses file1 and renames it to file.gz
gunzip file1.gz ===>> Decompresses file1.gz back to file
gzip -9 ===>> more compresor
gzip -d file1.gz
```

---

### bzip2

```bash
bzip2 ===>> better algorithm for compression than gzip
bzip2 file1 ===>> Compresses file1 and renames it to file.bz2
bunzip2 file1 Decompresses file1.gbz2 back to file
```

---

### xz

```bash
xz file1 ===>> Compresses file1 and renames it to file.xz
unxz file1 ===>> Decompresses file1.gxz back to file
xz -d ===>> unzip file  
```

---

### tar

```bash
tar -cvf file.tar file1 ===>> f = file
tar -xvf ===>> untar
tar -czvf myfile.tar.gz file1 ===>> z = gzip compres the file too
tar -xzvf ===>> extract
tar -cjvf myfile.tar.bz2 file1 ===>> j = gzip compres the file too
tar -xjvf ===>> extract  
tar -cf file.tar file ===>> Create tar named file.tar containing file
tar -xf file.tar ===>> Extract the file from file.tar
tar -czf file.tr.gz files ===>> Create a tar with Gzip compression
tar -xzf file.tar.gz ===>> Extract a tar using Gzip
tar -cjf file.tar.bz2 ===>> Create tar with Bzip2 compression
tar -xjf file.tar.bz2 ===>> Extract a tar using Bzip2
tar -Jcvf FILE_NAME.tar.xz ===>> make xz
tar -Jxvf FILE_NAME.tar.xz ===>> extract xz
```
