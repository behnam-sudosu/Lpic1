# grep

```bash
grep ===>> find patern in file or output command
grep ===>> searches string for special text, inside files ...
grep ===>> Searches for all the named processes, that matches with the pattern and, by default, returns their ID
grep PATTERN FILE_NAME ===>> Search for pattern in file
cat FILE_NAME | grep PATTERN
grep "yes" ssh_config ===>> find "yes" in file ssh_config
grep -i ===>> upercase & lowercase
grep -n ===>> put number
grep -c ===>> count line
grep -v ===>> line no have "yes"
grep -r ===>> find in folder too
grep -w ===>> just find "yes" not patern
grep -A 3 ===>> show 3 line next
grep -B 3 ===>> show 3 line before
grep -E ===>> like egrep
egrep "^stars" ===>> line start with "stars"
grep "bb$" ===>> find text bb at the end of line
grep a.b ===>> a at start and b at the end
grep ^...$ ===>> only show 3 character
ls /etc | grep passwd
command | grep pattern ===>> Search pattern in the output of a command
pgrep ===>> process id
```
