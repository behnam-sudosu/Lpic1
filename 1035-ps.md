# ps  

### ps ===>> show linux processe
	ps ===>> shows linux processes, list of programs that are running
    ps -a ===>> all user 
    ps -u ===>> user
    ps -x ===>> all process in linux os
    ps -aux ===>> show all process
    ps -aux | grep sleep
    ps -fl ===>> show nice process
    ps ===>> ps -aux | grep ssh ===>>> show ssh
    ps -ef ===>> more information PID & PPID
    ps -l ===>> list
    pstree ===>> show your process and dependency
    pgrep ===>> show process id
	pgrep sleep ===>> show process ID sleep
