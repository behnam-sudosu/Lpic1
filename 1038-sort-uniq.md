# sort & uniq commands

### sort

```bash
sort ===>> organize text in a file (numbers, alphabets)
sort FILE_NAME
sort -r FILE_NAME ===>> vica versa
sort -n ===>> sort digit
```

---

### uniq

```bash
uniq ===>> show only one
uniq -c ===>> show how many
uniq -d ===>> show repeated
uniq -u ===>> show not repeated
```
