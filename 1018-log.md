### journalctl

```bash
# show all log
journalctl

# show end
journalctl -f

# show details
journalctl -ex

journalctl -f | grep nginx

# only show sshd
journalctl -u sshd

# reload some program that is save in memory
journalctl daemon-realod
```

```bash
dmesg ===>> you can viewreal-time kernel messages, show all

```

### make logrotate

```bash
vim /etc/logrotate.conf

# make file for log
vim /etc/logrotate.d

# for example
vim nginx

	/var/log/dpkg.log {
	monthly
	rotate 12
	compress
	delaycompress
	missingok
	notifempty
	create 644 root root
	}
```
